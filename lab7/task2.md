# Конфижим для удобства

1. Где хранятся пользовательские и системные настройки подключения?  
    Пользовательские: `~/.ssh/config`, системные: `/etc/openssh/ssh_config`.

2. Что за файл options?  
    Файл options (SSH-конфиг клиента) содержит предустановленные параметры подключения к серверам.

3. Отредактируйте файл options так, чтобы можно было подключаться не вводя имя пользователя и порт  
    В `~/.ssh/config` добавлен блок с параметрами HostName, User, Port.

4. Назовите подключение удобным для вас способом  
    Создано подключение с именем `myserver`.

5. Проверьте работоспособность  
    Проверено: `ssh myserver` подключается без запроса имени пользователя и порта.
    ssh: connect to host 192.168.1.100 port 22: Connection timed out
