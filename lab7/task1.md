# Настраиваем

1. Какой по умолчанию используется порт для подключения?  
    По умолчанию SSH использует порт 22.

2. Можно ли его изменить? если да то как?  
    Да, можно изменить в файле конфигурации `/etc/ssh/sshd_config`, изменив строку `Port` и перезапустив службу `sshd`.

3. Какая служба отвечает за обработку запросов на подключения по ssh?  
    За обработку SSH-подключений отвечает служба sshd (OpenSSH server).

4. Какой файл конфигурации отвечает за его настройку?  
    Основной файл конфигурации SSH-сервера: `/etc/openssh/sshd_config`.
5. Попробуйте подключиться по ssh к предоставленному вам серверу
сделано 

6. Отредактируйте файл настроек так, чтобы была возможность подключиться как root  
    Создан файл `/etc/ssh/opensshd_config` с директивой `PermitRootLogin yes`.

7. Измените количество ошибок ввода пароля  
    Добавлена директива `MaxAuthTries 4` в конфиг.

8. Создайте пользователя ssh-user  
    Создан пользователь `ssh-user`, подключение: `ssh ssh-user@localhost`.

9. Ограничьте ему возможность подключения  
    Добавлена директива `DenyUsers ssh-user` в конфиг.

10. Как вы это сделали?  
     Через добавление директив в `/etc/openssh/sshd_config` и перезапуск службы.

11. Что хранится в файле known_hosts?  
     Отпечатки SSH-ключей серверов для проверки подлинности.
