# Журнальчики

1. Посмотрите журналы ssh  
   + `journalctl -u sshd` – показывает все логи SSH-сервера.

2. Выведите журналы в реальном времени  
   + `journalctl -f` – выводит логи в реальном времени (follow mode).

3. Выведите лог в реальном времени для службы sshd  
   + `journalctl -f -u sshd` – показывает логи SSH в реальном времени.

4. Можно ли без команды journalctl прочитать логи systemd?  
   + Да, логи systemd хранятся в `/var/log/journal/` и могут быть прочитаны напрямую.

5. Сколько будет 2-2?  
   + `expr 2 - 2` – результат: 0
