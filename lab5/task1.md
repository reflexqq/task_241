# Юниты

1. Что такое systemd юнит?  
    Systemd юнит – конфигурационный файл для управления службами в Linux.

2. Проверьте статус любого systemd юнита, какую информацию выводит эта команда?  
   `systemctl status sshd` – показывает состояние SSH-сервера: активно/неактивно, время работы, логи, PID процесса.

3. Попробуйте остановить сервис.  
    `systemctl stop sshd` – останавливает SSH-сервер.

4. Перезапустите его.  
    `systemctl restart sshd` – перезапускает SSH-сервер.

5. Удалите из автозагрузки.  
    `systemctl disable sshd` – отключает автозагрузку SSH-сервера.

6. Верните обратно.  
    `systemctl enable sshd` – включает автозагрузку SSH-сервера.

7. Что такое таймеры?  
    Таймеры – юниты systemd для запуска задач по расписанию, аналогично cron.
