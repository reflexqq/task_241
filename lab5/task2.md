# Пишем юниты

1. Создайте скрипт который создаёт папку, заполняет её файлами  
    Скрипт `/usr/local/bin/my_script.sh` создаёт папку `my_data`, файлы `file_1.txt` – `file_4.txt` с информацией о дате, версии ядра, имени компьютера и списке файлов домашнего каталога.

2. Создайте юнит который будет вызывать этот скрипт при запуске  
    Создан юнит `/etc/systemd/system/my_service.service`, проверка: `systemctl status my_service`.

3. Создайте таймер который будет вызывать выполнение каждые 5 минут  
    Создан таймер `/etc/systemd/system/my_service.timer`, активирован: `systemctl enable --now my_service.timer`.

4. От какого пользователя вызываются юниты по умолчанию?  
    По умолчанию юниты выполняются от пользователя root.

5. Создайте пользователя от имени которого будет выполняться ваш скрипт  
    Создан пользователь: `useradd -m script_user`.

6. Дополните юнит информацией о пользователе  
    В юнит добавлены строки: `User=script_user` и `WorkingDirectory=/home/script_user`.

7. Дополните ваш скрипт так, чтобы он всегда выполнялся в домашней папке  
    В начало скрипта добавлена команда: `cd ~`.
