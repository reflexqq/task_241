## Открываем iptables
1. Установите iptables
#sudo apt-get install iptables - установил
2. Проверьте осталась ли возможность подключения по ssh к вашему серверу
Да,осталось
3. Почему может пропасть такая возможность?
Подключение по SSH может пропасть, если:

iptables по умолчанию блокирует все входящие подключения
Правила не разрешают порт 206(SSH)
Были применены правила DROP или REJECT для входящих соединений
4. Откройте нужный порт на сервере чтобы восстановить подключение
sudo iptables -A INPUT -p tcp --dport 206 -j ACCEPT
5. Это будет udp или tcp порт?
tcp
## Сохраняем
6. Сохраняются ли записанные вами правила после перезагрузки?
Неа) правила iptables, добавленные через команды iptables, не сохраняются после перезагрузки системы.
7. Как их сохранить?
sudo iptables-save | sudo tee /etc/iptables/rules.v4 > /dev/null (без пояснений, иначе чет не шло)
