## 2. Где хранятся папки пользователей в системе?

Папки пользователей хранятся в директории /home/. Каждый обычный пользователь (не root) имеет свою подпапку с именем, соответствующим его логину.
## 3. Где домашняя папка суперпользователя?

Домашняя папка суперпользователя (root) находится по пути root. Это отдельная директория на корневом разделе
## 4. Где хранятся основные конфигурационные файлы в системе?

Основные конфигурационные файлы системы хранятся в директории etc (от англ. "et cetera" — и другие).
## 5. Что за папки /bin, /sbin, /usr/bin, /usr/sbin?

### Краткое описание:

- /bin – Основные команды, доступные всем пользователям. Необходимы при загрузке системы.
- sbin – Системные команды для администрирования (только для root). Требуются для загрузки и восстановления системы.
- /usr/bin – Пользовательские программы и приложения. Не нужны на ранних этапах загрузки.
- /usr/sbin – Системные программы для администраторов. Аналогично /usr/bin, но требуют прав root.
